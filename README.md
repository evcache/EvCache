# EvCache

EvCache offers a set of utilities and case studies for recovering and using
cache-related information inside VMs and CVMs.

Start with [the overview](./docs/1-overview.md). It includes the build steps,
setup notes, and links to the utility and case-study documentation.

The main documentation is organized as:

```text
docs/
  1-overview.md
  2-utilities.md
  3-cache-parameters.md
  4-gpa-hpa-debug.md
  5-graphing-and-data.md
  6-case-studies/
    1-lcas.md
    2-vcpc.md
```

After building, utilities such as `vev`, `vset`, `vpo`, `vcolor`, and `vtest`
are available under `build/`. See [EvCache utilities](./docs/2-utilities.md)
for usage.
